Online Install/Update from Eclipse
Follow the instructions from Install and verify
This is the recommended way to install/update the project
Offline Update Site Installation
Go to the "Code" section on GitHub IusCL repository, into "docs/update" folder (https://github.com/iuscl-ide/IusCL/tree/master/docs/update), and download the whole version folder (all "update"). Then in Eclipse go to "Help" menu, then "Install New Software..." and add the downloaded version folder as a local update site. Then proceed the same way as for the online installation.
